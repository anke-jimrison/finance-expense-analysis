# Finance & Expense Analysis

## Objective
To analyze financial and expense data to identify cost drivers, profitability trends, and budget optimization opportunities.

## Tools & Skills
- Excel
- SQL
- Python
- Power BI

## Dataset
Public financial dataset sourced from Kaggle.

## Key Insights
- Monthly expense trends
- Profit vs cost comparison
- High-spend categories

## Outcome
Provides clear financial insights to support better budgeting and cost control.
